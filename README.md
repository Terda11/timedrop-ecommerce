## 🏪 About the Store

**TIMEDROP** is a modern online watch store based in Rwanda, offering premium timepieces at competitive prices. The store targets young, style-conscious customers who appreciate quality watches — from classic leather-strap designs to bold modern sport watches.

> *"TimeDrop is where style meets time. We bring you the freshest watch drops — from classic designs to bold modern pieces — all in one place. Because the right watch changes everything."*

**Mission:** To make premium, stylish watches accessible to everyone in Rwanda and beyond, with a seamless and enjoyable online shopping experience.



## ✅ Features Implemented

### Pages
| Page | Description |
|---|---|
| 🏠 **Homepage** | Store name (TIMEDROP) + welcome message + hero banner |
| 📦 **Catalog (Product Page)** | Minimum 5 products with images, prices, and descriptions |
| 📞 **Contact Page** | Contact form with name, email, phone, and message fields |
| 🛒 **Cart** | Add-to-cart functionality with cart counter |

### Functional Features
- ✅ **Add to Cart** — Customers can add watches to their shopping cart
- ✅ **Order Form** — Customers fill in Name, Phone, and Address to place an order
- ✅ **Email Notifications** — Store owner receives an email notification for every order placed
- ✅ **Free Shipping** — Displayed on product pages
- ✅ **Multi-currency** — Store displays prices in RWF (Rwandan Franc)
- ✅ **Search** — Customers can search for products
- ✅ **Account** — Customer account creation and login
- ✅ **Responsive Design** — Works on mobile and desktop

## ⚠️ Challenges Faced

1. **Shopify Password Protection** — The free Shopify trial puts the store behind a password by default, making it inaccessible to the public. I had to disable the storefront password in settings to make the store publicly accessible.

2. **Product Image Quality** — Finding high-quality product images that matched the store's aesthetic required extra time and effort.

3. **Currency Setup** — Configuring the store to display prices in Rwandan Francs (RWF) instead of USD required adjusting the market and currency settings in Shopify.

4. **GitHub & Markdown** — Learning how to write proper Markdown documentation and push files to a GitHub repository was a new skill that required practice.

---

## 📚 Lessons Learned

1. **No-Code platforms are powerful** — Shopify allows building a professional e-commerce store without writing a single line of code, which is very impressive.

2. **GitHub is essential for developers** — I learned how to create a repository, write Markdown documentation, and manage project files using GitHub.

3. **User Experience matters** — Designing pages that are easy to navigate and visually appealing is just as important as the products themselves.

4. **E-commerce has many moving parts** — Running an online store involves product management, payment setup, shipping, customer contact, and marketing all working together.

5. **Documentation is part of the project** — Writing a clear README.md helped me reflect on the work I did and communicate it professionally.
